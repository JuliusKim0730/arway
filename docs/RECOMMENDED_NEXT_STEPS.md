## 🎯 즉시 추천 작업 (우선순위 순)

### 1. 경로 히스토리 기능 📜
**우선순위**: 🔴 높음  
**예상 시간**: 4-6시간  
**왜 추천하는가**: 사용자가 과거 네비게이션 기록을 확인하고 재사용할 수 있어 편의성이 크게 향상됩니다.

**구현 내용**:
- 히스토리 화면 추가 (`/ar-nav/history`)
- 과거 네비게이션 세션 목록 표시
- 목적지 이름, 날짜, 소요 시간 표시
- 클릭 시 해당 목적지로 바로 재시작
- 통계 정보 (총 횟수, 평균 시간)

**사용자 가치**: ⭐⭐⭐⭐⭐ (매우 높음)
