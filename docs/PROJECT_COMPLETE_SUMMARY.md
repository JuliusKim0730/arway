## 🚀 실행 방법

### 빠른 시작

```bash
# 1. PostgreSQL 실행
cd new_challange
docker-compose up -d postgres

# 2. 백엔드 실행
cd backend
..\venv\Scripts\python.exe -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 3. 프론트엔드 실행
cd frontend
npm run dev
```

### 접속 URL
- **프론트엔드**: http://localhost:3000/ar-nav
- **백엔드 API**: http://localhost:8000
- **API 문서**: http://localhost:8000/docs
