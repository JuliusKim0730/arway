## 🧪 테스트 시나리오

### 시나리오 1: 실외 → 실내 전환 테스트

1. **시작 위치**: 강남역 근처 (37.4980, 127.0278)
2. **목적지**: 테스트 백화점 내부 스타벅스
3. **예상 동작**:
   - 실외 네비게이션 시작
   - 건물 근처 도착 시 지오펜스 진입 감지

   - AR 행동 지시 표시

## 🔧 API 테스트

### 지오펜스 조회
```bash
curl "http://localhost:8000/api/v1/geofences?lat=37.4979&lng=127.0276&radius=1000"
```

### 실내 맵 조회
```bash
# 건물 ID 확인 후
curl "http://localhost:8000/api/v1/indoor-maps?building_id={building_id}"
```

### POI 조회
```bash
# 실외 POI
curl "http://localhost:8000/api/v1/pois?lat=37.4979&lng=127.0276&radius=100"

# 실내 POI
curl "http://localhost:8000/api/v1/pois?indoor_map_id={indoor_map_id}"
```
