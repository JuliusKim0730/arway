## ✅ 발견 및 수정된 문제

### 1. 프론트엔드 빌드 오류

#### 문제 1: Import 경로 오류
**파일**: `frontend/lib/api/scq.ts`  
**오류**: `Module not found: Can't resolve './client'`  
**원인**: 잘못된 import 경로  
**수정**: 
```typescript
// 수정 전
import { apiGet, apiPost } from './client';

// 수정 후
import { apiGet } from '../apiClient';
```

#### 문제 2: 누락된 Analytics 이벤트
**파일**: `frontend/lib/analytics.ts`  
**오류**: `AnalyticsEvents.INDOOR_MODE_ACTIVATED` 정의되지 않음  
**수정**: 
```typescript
export const AnalyticsEvents = {
  // ... 기존 이벤트들
  INDOOR_MODE_ACTIVATED: 'indoor_mode_activated',
} as const;
```
