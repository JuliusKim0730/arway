## 🚀 서버 실행 방법

### 방법 1: Python 모듈로 직접 실행 (권장)

```powershell
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 방법 2: 제공된 스크립트 사용

#### Windows (배치 파일)
```cmd
cd backend
start_server.bat
```

#### PowerShell
```powershell
cd backend
.\start_server.ps1
```

---

## 🔧 문제 해결

### 문제: 'uvicorn' 용어가 인식되지 않음

#### 원인
- 가상 환경이 활성화되지 않음
- uvicorn이 전역적으로 설치되지 않음

#### 해결 방법

**방법 1: Python 모듈로 실행 (가장 안전)**
```powershell
python -m uvicorn app.main:app --reload
```

**방법 2: 가상 환경 활성화 후 실행**
```powershell
# 가상 환경 활성화
.\venv\Scripts\Activate.ps1
# 또는
.\.venv\Scripts\Activate.ps1

# 그 다음 uvicorn 실행
uvicorn app.main:app --reload
```

**방법 3: uvicorn 설치 확인**
```powershell
pip install uvicorn[standard]
```

## 🧪 API 테스트

서버가 실행되면 다음 명령어로 테스트:

```powershell
# 헬스 체크
curl http://localhost:8000/health

# 건물 목록
curl http://localhost:8000/api/v1/buildings

# 지오펜스 조회
curl "http://localhost:8000/api/v1/geofences?lat=37.4979&lng=127.0276&radius=1000"

# POI 조회
curl "http://localhost:8000/api/v1/pois?lat=37.4979&lng=127.0276&radius=100"
```
