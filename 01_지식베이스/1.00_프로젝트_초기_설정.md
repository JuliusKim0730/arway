# 1.00 프로젝트 초기 설정 및 환경 구성

## 워크플로우 개요

프로젝트의 기본 인프라와 개발 환경을 설정하는 단계입니다.

## 목표

- 로컬 개발 환경 구축
- 프로젝트 구조 생성
- 필수 도구 설치 및 설정
- 데이터베이스 초기 설정

## 단계별 작업

### 1.1 프로젝트 구조 생성

```
arway-lite/
├── frontend/              # Next.js 프론트엔드
│   ├── app/
│   ├── components/
│   ├── hooks/
│   ├── lib/
│   └── public/
├── backend/              # FastAPI 백엔드
│   ├── app/
│   │   ├── api/
│   │   ├── models/
│   │   ├── schemas/
│   │   └── services/
│   ├── alembic/
│   └── tests/
├── admin/                # Admin 관리자 화면 (Next.js)
│   └── app/
├── database/             # 데이터베이스 스크립트
│   ├── migrations/
│   └── seeds/
└── docker-compose.yml    # 로컬 개발용 Docker Compose
```

### 1.2 Frontend 초기 설정

#### 1.2.1 Next.js 프로젝트 생성
```bash
cd frontend
npx create-next-app@latest . --typescript --tailwind --app
```

#### 1.2.2 필수 패키지 설치
```bash
npm install geolib
npm install zustand  # 상태 관리
npm install axios     # API 통신
npm install -D @types/geolib
```

#### 1.2.3 환경 변수 설정
`.env.local` 파일 생성:
```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

### 1.3 Backend 초기 설정

#### 1.3.1 Python 가상환경 생성
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

#### 1.3.2 필수 패키지 설치
```bash
pip install fastapi uvicorn sqlalchemy psycopg2-binary alembic python-dotenv pydantic
```

#### 1.3.3 프로젝트 구조 생성
```
backend/
├── app/
│   ├── __init__.py
│   ├── main.py          # FastAPI 앱 진입점
│   ├── config.py        # 설정 관리
│   ├── database.py      # DB 연결
│   ├── api/
│   │   └── v1/
│   │       ├── __init__.py
│   │       ├── users.py
│   │       ├── destinations.py
│   │       └── sessions.py
│   ├── models/          # SQLAlchemy 모델
│   ├── schemas/         # Pydantic 스키마
│   └── services/        # 비즈니스 로직
├── alembic.ini
└── requirements.txt
```

#### 1.3.4 환경 변수 설정
`.env` 파일 생성:
```
DATABASE_URL=postgresql://arway_user:password@localhost:5432/arway_lite
SECRET_KEY=your-secret-key-here
```

### 1.4 데이터베이스 설정

#### 1.4.1 PostgreSQL 설치 (로컬)
- Windows: PostgreSQL 공식 설치 프로그램
- Mac: `brew install postgresql@15`
- Linux: `sudo apt-get install postgresql-15`

#### 1.4.2 데이터베이스 생성
```sql
CREATE DATABASE arway_lite;
CREATE USER arway_user WITH PASSWORD 'password';
GRANT ALL PRIVILEGES ON DATABASE arway_lite TO arway_user;
```

#### 1.4.3 Alembic 초기화
```bash
cd backend
alembic init alembic
```

### 1.5 Docker Compose 설정 (선택사항)

`docker-compose.yml` 생성:
```yaml
version: '3.8'

services:
  postgres:
    image: postgres:15-alpine
    environment:
      POSTGRES_DB: arway_lite
      POSTGRES_USER: arway_user
      POSTGRES_PASSWORD: password
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data

volumes:
  postgres_data:
```

### 1.6 Admin 초기 설정

#### 1.6.1 Admin 프로젝트 생성
```bash
cd admin
npx create-next-app@latest . --typescript --tailwind --app
```

#### 1.6.2 Admin 전용 패키지 설치
```bash
npm install recharts  # 차트 라이브러리
npm install react-table  # 테이블 컴포넌트
```
